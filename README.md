# Wazuh SIEM Implementation
### Project Overview
Yeh project mein maine **Wazuh** (Open Source SIEM) install kiya hai jo real-time mein logs monitor karta hai, attacks detect karta hai aur beautiful dashboard deta hai.

### What I Built & Learned
- Wazuh Manager + Agent successfully installed on Linux
- Configured it to monitor system logs and network activity
- Created custom rules for detecting brute-force attacks
- Set up Kibana dashboard for visualization
- Simulated attacks and saw real-time alerts

### Tools & Technologies Used
- **Wazuh** (SIEM + XDR)
- Elasticsearch & Kibana
- Kali Linux / Ubuntu
- Log Analysis & Threat Detection

### Key Features
- Real-time intrusion detection
- Vulnerability monitoring
- File integrity monitoring
- Active response capabilities

### Conclusion
Is project ne mujhe **SOC Analyst** ka real work experience diya. Ab main better samajh gaya hoon ki companies mein Security Operations Center kaise kaam karta hai.
