Wazuh SIEM Implementation
Project Overview
In this project, I successfully installed and configured Wazuh (an Open Source SIEM solution) that monitors logs in real-time, detects attacks, and provides a visually appealing dashboard for security monitoring.
What I Built & Learned

Successfully installed Wazuh Manager and Agent on Linux
Configured it to monitor system logs and network activity
Created custom rules for detecting brute-force attacks
Set up Kibana dashboard for data visualization
Simulated real attacks and observed real-time alerts

Tools & Technologies Used

Wazuh (SIEM + XDR)
Elasticsearch & Kibana
Kali Linux / Ubuntu
Log Analysis & Threat Detection

Key Features

Real-time intrusion detection
Vulnerability monitoring
File integrity monitoring
Active response capabilities

Conclusion
This project gave me practical, real-world experience similar to that of a SOC Analyst. It helped me deeply understand how Security Operations Centers (SOC) function in companies and how security teams monitor, detect, and respond to threats.
