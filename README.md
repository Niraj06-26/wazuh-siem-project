# Wazuh SIEM Implementation

## Project Overview

In this project, I successfully deployed and configured **Wazuh**, an open-source Security Information and Event Management (SIEM) platform, to monitor system activities, detect security threats, and provide real-time security visibility through an interactive dashboard.

## What I Built & Learned

* Installed and configured Wazuh Manager and Wazuh Agent on Linux systems.
* Configured log collection and monitoring for system and network activities.
* Created and customized detection rules to identify brute-force attacks and suspicious activities.
* Integrated and configured the Kibana dashboard for log visualization and security monitoring.
* Simulated security attacks and analyzed real-time alerts generated by Wazuh.
* Gained hands-on experience in security event monitoring, log analysis, and threat detection.

## Tools & Technologies Used

* Wazuh (SIEM & XDR Platform)
* Elasticsearch
* Kibana
* Kali Linux
* Ubuntu Linux
* Log Analysis
* Threat Detection & Monitoring

## Key Features

* Real-time intrusion detection and alerting
* Vulnerability monitoring and assessment
* File Integrity Monitoring (FIM)
* Active response and threat mitigation capabilities
* Centralized log management and analysis

## Conclusion

This project provided practical exposure to Security Operations Center (SOC) activities and helped me understand how organizations monitor, detect, and respond to security threats using SIEM solutions. It strengthened my knowledge of log analysis, incident detection, and security monitoring processes used in real-world cybersecurity environments.
